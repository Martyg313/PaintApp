<h1>Paint App (WPF)</h1>

<h2>Description</h2>
A simple paint app built with .NET WPF and SkiaSharp graphics library. Inspired by Microsoft Paint, this app is supposed to be a light weight graphics editor. The tools include a brush, select, shape, flood, and text tool. The user may also save and open .png files to edit.
<br/>

<h2>Languages</h2>

- <b>C#</b>
